Present Indefinite Tense

Structure:

subject + verb + extension

For example:

  1. I eat rice
  2. I miss you

Present Continuous Tense

Structure:

Subject + am/is/are + verb + ing + extension

For example:

  1. She is making tea
  2. I am eating rice

Present Perfect Tense

Structure:

Subject + have/has + (v3) + extension

For example:

  1. I have read the book
  2. We have solved the problem

Present Perfect Continuous Tense

Structure:

  1. Subject + have been/has been + verb + ing + -----
  2. Subject + have been/has been + verb + ing + since/for + -----

For example:

  1. I have been reading this book since morning
  2. He has been working for two hours
